# Charlie Sans Font Files

Place your Charlie Sans font files in this directory.

## Required Files:

1. **charlie-sans-regular.woff2** (or .woff, .ttf) - Normal weight (400)
2. **charlie-sans-bold.woff2** (or .woff, .ttf) - Bold weight (700)

## File Naming:

The font files should be named exactly as:
- `charlie-sans-regular.woff2` (or `.woff`, `.ttf`)
- `charlie-sans-bold.woff2` (or `.woff`, `.ttf`)

## Where to Get the Font:

You can download Charlie Sans from:
- FontLot: https://fontlot.com/charlie-sans-serif-font/
- FontForFree: https://fontforfree.com/charlie-font/
- CufonFonts: https://www.cufonfonts.com/font/charlie

## Converting Font Files:

If you have `.ttf` or `.otf` files, you can convert them to `.woff2` format using online tools like:
- https://cloudconvert.com/ttf-to-woff2
- https://convertio.co/ttf-woff2/

## Note:

Until you add the font files, the application will use the fallback font (Arial).
